Face evolver using GAN
  • by Filipe Zappe
Based on GAN showcase

We STRONGLY reccomend that you use a GPU-powered computer to use this.
Click in the generate button to populate the initial generation.
Click the faces you want to select to the next generation and click the generation button when done. Note that selected faces get an small green square on the top left.
The float array used to generate the clicked face appears on the textarea below. Yoou can copy and save it.
You can later click on a face, paste the array on the textarea and click the SET button. This will set the clicked face to the one generated by the pasted array.
